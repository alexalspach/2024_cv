3D Printed Soft Skin for Safe Human-Robot Interaction
Kim, J., Alspach, A., Yamane, K.
Design and Hardware Development, IROS 2015, Hamburg, Germany, Sept. 2015, Disney Research.

Lead designer and developer of soft skin module with a built-in airtight cavity in which air pressure can be sensed. A pressure feedback controller is implemented on a robotic system using this module for contact sensing and gentle grasping. Modules are fabricated using a multimaterial 3D printer.
